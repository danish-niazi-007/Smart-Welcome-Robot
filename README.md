# 🤖 AI Exhibition Smart Greeter Bot

An advanced AI-powered Exhibition Robot Face UI built using **Python, CustomTkinter, and Computer Vision concepts**.  
This smart robot detects visitors, identifies gender/group scenarios, changes facial expressions dynamically, and greets users with animated UI and voice interaction in real-time.

---

# 🚀 Features

- 🎭 Dynamic Robot Face Animation
- 👀 Real-Time Visitor Detection
- 🧠 Gender-Based Smart Greeting System
- 👨 Single Male Detection
- 👩 Single Female Detection
- 👥 Group Detection Support
- 🌈 Automatic Theme & Color Switching
- 🔊 Text-To-Speech Greeting
- ✨ Smooth Typewriter Greeting Animation
- ⚡ Real-Time UI Updates
- 🛰️ AI Scanning Effects & Animated Particles
- 🕒 Live Digital Clock
- 📊 Detection Statistics Panel

---

- Python

# 🧠 How It Works

1. Camera starts scanning for visitors  
2. AI detector analyzes faces  
3. Gender/group scenario is detected  
4. Robot expression changes automatically  
5. Greeting text appears with typewriter effect  
6. TTS welcomes the visitor  
7. Live statistics update instantly  

---

# 🎨 Supported Detection Scenarios

| Scenario | UI Behavior |
|----------|-------------|
| No Visitor | Idle Scanning Mode |
| Single Male | Blue Theme + Smile |
| Single Female | Pink Theme + Smile |
| Male Group | Neon Blue Group Mode |
| Female Group | Pink Group Mode |
| Mixed Group | Purple Mixed Theme |
